# Chunchana-2020
This repository contains files of the website for Chunchana-2020(Annual college festival of AIT, Chikkamagaluru.)
* An event management Django web application, where students can register for various events, the applications are to be monitored by the faculties.
* Participants can be eliminated, or promoted to next round (Control Dashboard).
* Admin panel for the judges to enter the points.
* Automatic Winner announcement based on the points (to avoid any conflicts).
